### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/fullstack-academy.git
cd fullstack-academy
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Open your browser**
```
http://localhost:5173
```
